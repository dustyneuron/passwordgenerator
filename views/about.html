About

I was inspired to build the Password Generator by the linux command pwgen, and by xkcd.com/936/.

Each of the languages uses a wordlist of around 40,000 words, so three words are enough to give a strong password. The downside is that some of the words are so rare only serious Scrabble players are likely to appreciate them.

Check out the source code on github.
